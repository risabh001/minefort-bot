```markdown
# Minecraft Bot

A 24/7 running Minecraft bot for the server crossstraight.minefort.com.

## Features
- Automatically reconnects if disconnected
- Responds to basic commands (!ping, !help, !time)
- Runs continuously using Railway

## Setup
1. Clone this repository
2. Install dependencies: `npm install`
3. Run locally: `npm start`

## Deployment
This bot is deployed on Railway for 24/7 uptime.

### Environment Variables
- MC_HOST: Minecraft server host (default: crossstraight.minefort.com)
- MC_PORT: Minecraft server port (default: 25565)
- MC_USERNAME: Bot username (default: Bot)
- MC_AUTH_TYPE: Authentication type (default: offline)
```
